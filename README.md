# 💍 Digital Wedding Planner

Welcome to the repository of my digital wedding planner landing page! This project is designed to help couples organize their dream wedding seamlessly, tracking everything from budgets and guest lists to timelines and vendor management in one clean, responsive dashboard.

### ✨ Key Features
* **Interactive Dashboard:** All-in-one view of wedding preparation progress.
* **Budget & Expense Tracker:** Smart budget allocator and payment tracker.
* **Guest List & RSVP Manager:** Keep track of invitations, RSVPs, and seating arrangements.
* **Countdown & Timeline:** Step-by-step wedding checklist based on the big day.

### 🛠️ Built With
* HTML
* CSS
* Javascript
